# 🌍 Người Trái Đất – Plant Pot E-Commerce Website

![Laravel](https://img.shields.io/badge/Laravel-10-red)
![PHP](https://img.shields.io/badge/PHP-8-blue)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange)
![License](https://img.shields.io/badge/License-Education-green)

**Người Trái Đất** là một website thương mại điện tử chuyên kinh doanh **chậu cây, cây cảnh và phụ kiện trồng cây**.  
Hệ thống cho phép người dùng tìm kiếm sản phẩm, đặt hàng trực tuyến và quản lý cửa hàng thông qua trang quản trị.

Dự án được phát triển bằng **Laravel (PHP) + MySQL** với frontend **HTML/CSS/JavaScript thuần**.

#  Main Features

##  Customer Features

### Product Catalog
- Hiển thị danh mục sản phẩm
- Xem chi tiết sản phẩm

### Product Search & Filter
Người dùng có thể:
- tìm kiếm sản phẩm
- lọc theo loại
- lọc theo giá

### User Authentication
- đăng ký tài khoản
- đăng nhập / đăng xuất
- cập nhật thông tin cá nhân

### Shopping Cart
- thêm sản phẩm vào giỏ hàng
- cập nhật số lượng
- xóa sản phẩm

### Checkout
- nhập thông tin giao hàng
- xác nhận đơn hàng
- tạo đơn hàng

### Order History
- xem danh sách đơn hàng
- xem chi tiết đơn hàng
